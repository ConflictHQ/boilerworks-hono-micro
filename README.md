# Boilerworks Hono Micro

> Edge template -- Hono API microservice on Cloudflare Workers. API-key auth, no user auth. No frontend.

**Status:** Planned

Hono-based edge API microservice running on Cloudflare Workers. API-key authentication, no user auth overhead. Ultra-low latency via edge deployment, no cold starts, no Docker in production. Ideal for lightweight APIs, webhook handlers, proxy services, and edge data processing.

**Edge template** -- production deployment targets Cloudflare Workers, not Docker.
